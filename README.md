## 📊 About The Project

Terminal Portfolio is an innovative, command-line styled web application showcasing Sumanth Kumar's professional journey, skills, and projects. Built with Next.js and Tailwind CSS, it offers a highly interactive, retro terminal experience that mimics a classic Windows command prompt complete with CRT scanlines and dynamic typing interactions.

### ✨ Key Features

- **💻 Authentic Terminal Interface** - Genuine command-line experience with CRT scanlines, retro colors, and accurate prompt behaviors.
- **🔍 Interactive Commands** - Explore portfolio details through terminal commands: `about`, `skills`, `projects`, `experience`, `education`, and `contact`.
- **👋 Dynamic Greetings** - Hidden conversational interactions (try typing `hi` or `hello`!).
- **📥 Download CV** - A dedicated `download` command featuring an authentic 3-second terminal loading bar before serving the PDF resume.
- **📱 Responsive Design** - Terminal output gracefully scales and wraps, providing a perfect mobile-first layout without breaking ASCII art structures.
- **🚀 Exit Animation** - Unique multi-stage exit sequence featuring a binary matrix drop and goodbye screens.

## 🚀 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository
   ```sh
   git clone https://github.com/Skarycloud/terminal-portfolio.git
   ```
2. Navigate into the project directory
   ```sh
   cd terminal-portfolio
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. Start the development server
   ```sh
   npm run dev
   ```
5. Open your browser to view the application at `http://localhost:3000`
